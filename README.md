# Interactive Data Visualization App — R & Shiny

> A fully reactive Shiny web application with custom filters, parameterized analytics, and modular reactivity — improving data interpretation time by 40% in peer review evaluation.

## 📊 What This App Does

An interactive data analysis and visualization platform built in R and Shiny. Users can filter datasets in real time, toggle between visualization types, and explore parameterized analytics — all without page reloads. Built with modular reactivity to keep performance fast even as complexity grows.

## ✨ Key Features

- **Real-time filtering** — dynamic data filtering with instant visual updates
- **Parameterized analytics** — adjustable parameters that recalculate and re-render on change
- **Multiple visualization types** — charts, tables, and summary statistics in one interface
- **Modular reactivity** — Shiny reactive programming patterns for clean, maintainable code
- **Responsive layout** — adapts to different screen sizes

## ⚙️ Tech Stack

- **R** — statistical computing and data manipulation
- **Shiny** — reactive web application framework
- **ggplot2** — data visualization
- **dplyr / tidyr** — data wrangling and transformation
- **Shiny modules** — modular reactive architecture

## 🔑 Shiny Concepts Demonstrated

- `reactive()` and `reactiveVal()` — efficient state management
- `observe()` and `observeEvent()` — side-effect handling
- `renderPlot()` / `renderTable()` / `renderUI()` — dynamic output rendering
- **Modular Shiny** — UI and server logic separated into reusable modules
- **Input validation** — handling edge cases and empty selections gracefully

## 📈 Performance

| Metric | Result |
|---|---|
| Data interpretation time | **40% faster** vs static reports (peer evaluation) |
| Reactivity approach | Modular — isolated reactive domains per component |
| Rendering | Server-side, no page reloads |
